1. 리액트와 리액트 돔을 설치한다.

npm i react react-dom

react-dom은 리액트를 html에 넣어주는 역할을 한다.

2. 웹팩을 설치한다.

npm i -D webpack webpack-cli

3. 바벨을 설치한다.

npm i -D @babel/core @babel/preset-env @babel/preset-react babel-loader

4. 각종 설정을 한다.

5. 웹팩을 빌드한다.

npx webpack

개발 서버 설정

6. 개발 서버를 설치한다.

npm i -D webpack-dev-server
